📋 配置参数说明

参数类型默认值说明
fileUploadBooleanfalse是否启用文件上传功能
fileUploadURLString""文件上传的服务器接口 URL
fileFormatsArray["doc","docx","pdf","txt","zip","rar","xls","xlsx","ppt","pptx","mp4","mp3"]允许上传的文件格式列表

🔧 PHP 后端配置参考

// php/upload.php 关键配置
$maxSizeMap = [
    XfEditorUploader::TYPE_FILE => 1024 * 20,    // 附件最大 20MB
];

$allowedFormats = [
    XfEditorUploader::TYPE_FILE => [
        'zip', 'rar', '7z', 'tar', 'gz',
        'doc', 'docx', 'xls', 'xlsx', 'ppt', 'pptx',
        'pdf', 'txt', 'md', 'csv',
        'mp3', 'wav', 'ogg',
    ],
];
