v1.17.9 新增功能速览 NEW

  • 🙈 Hidden 隐藏代码块：```(hidden) ```(hidden.class#id) 语法，渲染为 display:none!important 的隐藏 pre 标签，配合 iframe:pre 可「阅后即焚」式预览
  • 🎨 代码块 class/id 属性扩展：```(.class) ```(#id) ```(.cls#id) 语法，为 pre 标签设置自定义属性
  • 🖱 iframe:pre 悬浮预览：[查看代码](tooltip:iframe:pre#id) 提取页面 pre 内容以深色主题展示
  • 📊 表格空单元格修复：修复 marked.js v0.3.3 Bug，最后一列为空时自动补全
  • 🛡 内存安全增强：Blob URL 自动释放、空值安全检查、选择器错误防护
  • 🖼️ Tooltip 尺寸修复：图片/iframe 显式宽高精确限制 + object-fit:contain，max-width 从 360px 放宽至 90vw
  • 📄 getHTML 增强：输出完整自包含 HTML 文档，含内联 Tooltip 完整样式，不依赖外部 CSS/JS
  • 上标/下标：^上标^ ^^下标^^ <<下标>^<上标>> 语法，行内上下角标及组合
  • 字体大小：!32 文字! 语法，自定义字号 8-200px
  • 脚注：[^脚注名] 引用 + [^脚注名]: 定义，文末自动汇总
  • ECharts 图表：柱状图、折线图、饼图、雷达图、漏斗图，```echarts 代码块
  • Tabs 标签页：[[tabs]]...[[/tabs]] 语法，支持内部嵌套代码块、表格、图表等
  • 多列排版：[[columns:N]]...[[/columns]] 报纸式多栏布局
  • 栅格化布局 NEW：[[row]]...[[col:N]]...[[/row]] Bootstrap 风格 10 栏栅格
  • 悬浮提示：文本/图片/iframe/HTML选择器/iframe:pre 五种类型悬浮提示
  • 文本对齐：Unicode 行内对齐，支持居中/左/右/两端
  • 视频/附件：[[video]] 和 [[file]] 标签
  • 图片尺寸：![alt](url)<W,H> 语法 + 拖拽调整（精准追踪同图多次出现）
  • 草稿暂存：浏览器端自动保存，防止误刷新丢失内容
  • 字帖：田字格/米字格/拼音格三种字帖，SVG 辅助线
  • 🔗 链接 target 语法 NEW：[文本](url){target=_blank} 支持新窗口/当前窗口跳转，工具栏链接对话框支持 target 选择
