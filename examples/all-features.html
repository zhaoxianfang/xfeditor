Editor.md v1.7.0 全部功能示例

All Features Demo — 本示例演示了 Editor.md 的所有功能与特性

v1.7.0 新增功能速览

  • ECharts 图表：支持柱状图、折线图、饼图、雷达图、漏斗图，使用 ```echarts 代码块语法
  • Tabs 标签页：使用 [[tabs]] ... [[/tabs]] 标签语法创建可交互标签页，内部支持代码块、表格、图表等任意 Markdown 内容
  • 多列排版：使用 [[columns:N]] ... [[/columns]] 标签语法实现报纸式 N 栏排版
  • 文本对齐：工具栏按钮直接插入 Unicode 行内对齐 ⁑⁑内容⁑⁑（居中）、⁑⁖内容⁖⁑（左）、⁑⠕内容⠕⁑（右）、⁑⁛内容⁛⁑（两端），适用于标题、段落、列表等任意行内位置
  • 图片尺寸：使用 ![alt](url)<W,H> 语法设定图片尺寸，预览区拖拽后左侧同步更新
  • 悬浮提示：链接使用 [文本](tooltip:内容)、[文本](tooltip:image:图片URL)、[文本](tooltip:iframe:页面URL)；图片使用 ![alt](url "tooltip:内容")
  • 浏览器草稿暂存：配置 draftAutoSave: true 自动暂存，防止误刷新丢失内容
  • Unicode 行内对齐：使用 ⁑⁑内容⁑⁑、⁑⁖内容⁖⁑、⁑⠕内容⠕⁑、⁑⁛内容⁛⁑ 实现行内对齐
  • Insert 插入下拉工具栏：将代码、预格式化文本、代码块、表格、日期时间、HTML 实体、分页符等归入统一的 insert 下拉菜单，工具栏更简洁
  • 文件上传：支持配置允许上传的文件格式，工具栏一键插入文件链接，同时支持填写 URL 和本地上传
  • 视频插入：支持 [[video]] 标签和 <video> 标签两种方式嵌入视频
  • 附件链接：支持 [[file]] 标签创建带文件类型图标的附件下载链接
