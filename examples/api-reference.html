初始化

基本用法

var editor = editormd("editor-id", {
    width: "100%",
    height: 640,
    path: "./lib/"
});

第一个参数为容器 ID，第二个参数为配置对象，返回编辑器实例。

图片上传接口

请求方式

POST 表单提交，enctype="multipart/form-data"

请求参数

参数类型必填说明
editormd-image-fileFile是图片文件
guidString否唯一标识

成功返回

{"success":1,"message":"上传成功！","url":"https://example.com/uploads/image.png"}

失败返回

{"success":0,"message":"错误提示信息"}

跨域上传

跨域时服务器返回 302 跳转到 callback 页面，URL 参数包含：

?success=1&message=xxx&url=https://example.com/img.png&dialog_id=xxx&temp=xxx

文件上传接口（v1.7.0 新增）

请求方式

POST 表单提交，enctype="multipart/form-data"

请求参数

参数类型必填说明
editormd-file-fileFile是文件
guidString否唯一标识

成功返回

{"success":1,"message":"上传成功！","url":"https://example.com/uploads/file.pdf","name":"file.pdf"}

服务端必须返回 success、url 和 name 字段，其中 name 会自动回填到文件名称输入框。

失败返回

{"success":0,"message":"错误提示信息"}
