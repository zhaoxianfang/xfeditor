📄 [[page:AN]] 纸张页面语法

以标准纸张尺寸展示内容 · 自动换行 · 超页高分页 · 支持嵌套

📝 Markdown 源码

💡 使用提示：
1. [[page:A4]]...[[/page]] 以 A4 纸张尺寸（794×1123px）展示内容
2. [[page:AN]]...[[/page]] 以 AN 纸张尺寸展示（等同于 A4，794×1123px）
3. [[page:A5]]...[[/page]] 以 A5 纸张尺寸（559×794px）展示内容
4. 支持 A0 ~ A8、AN、LETTER、LEGAL 多种纸张规格
5. 页头页脚：[[page:A4 header="标题" footer="第 {page} 页 / 共 {total} 页"]]
6. 页面内可以嵌套 [[tabs]]、[[columns]]、[[copybook]] 等语法
7. 使用 [========] 分页符可以在页面内手动分页
8. 内容超出页面高度时自动分割为多页，连续页面间距 4px
9. 配置项 pageBlock: false 可以禁用此语法
10. 支持 @media print 打印样式
11. 正则匹配支持大小写不敏感，[[page:a4]] 和 [[page:A4]] 均有效

"pagebreak", // 分页符按钮 "page" // 页面尺寸下拉菜单 ]; } }); ```
