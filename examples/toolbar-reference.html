基础编辑

图标 / 名称功能描述用法说明
撤销 撤销上一步编辑操作 快捷键 Ctrl+Z，或点击工具栏按钮
重做 恢复被撤销的操作 快捷键 Ctrl+Y，或点击工具栏按钮
粗体 将选中文本设为粗体 选中文字后点击，生成 **粗体**
删除线 为选中文本添加删除线 选中文字后点击，生成 ~~删除线~~
斜体 将选中文本设为斜体 选中文字后点击，生成 *斜体*
引用 插入引用块 选中文字后点击，生成 > 引用文本
Aa 首字母大写 将每个单词首字母转为大写 选中文字后点击，自动转换格式
转大写 将选中文本全部转为大写 选中文字后点击，自动转换格式
a 转小写 将选中文本全部转为小写 选中文字后点击，自动转换格式

格式排版

图标 / 名称功能描述用法说明
H1 标题 插入 H1~H6 标题 下拉选择 H1~H6，生成 # 标题 ~ ###### 标题
无序列表 插入无序列表 点击生成 - 列表项，支持多级嵌套
有序列表 插入有序列表 点击生成 1. 列表项
分隔线 插入水平分隔线 点击生成 --- 或 ***
左对齐 插入 Unicode 左对齐标记 选中文字后点击，生成 ⁑⁖内容⁖⁑
居中对齐 插入 Unicode 居中对齐标记 选中文字后点击，生成 ⁑⁑内容⁑⁑
右对齐 插入 Unicode 右对齐标记 选中文字后点击，生成 ⁑⠕内容⠕⁑
两端对齐 插入 Unicode 两端对齐标记 选中文字后点击，生成 ⁑⁛内容⁛⁑
拼音标注 为汉字添加拼音注音 选中文字后点击，生成 {文字 | pinyin}

插入内容

图标 / 名称功能描述用法说明
普通链接 插入超链接 弹出对话框填写 URL 和标题，生成 [文本](url)
引用链接 插入行内引用链接 弹出对话框填写文本、URL 和标题，生成 [文本](url "标题")
图片 插入图片，支持 URL 和本地上传 弹出对话框填写 URL 或选择本地图片上传，支持尺寸语法 <W,H>
文件 插入文件链接，支持 URL 和本地上传 弹出对话框填写 URL 或选择本地文件上传，生成 [文件名](url)
插入 下拉 统一的插入内容下拉菜单 包含代码、预格式化文本、代码块、表格、日期时间、HTML 实体、分页符
行内代码 插入行内代码 选中文字后点击，生成 `code`
预格式化文本 插入预格式化文本块 弹出对话框选择语言，生成缩进代码块
代码块 插入带语法高亮的 fenced 代码块 弹出对话框选择语言，生成 ```lang ... ```
表格 插入 Markdown 表格 弹出对话框设置行列数，自动生成表格语法
日期时间 插入当前日期时间 点击自动生成当前日期时间文本，如 2026-06-03 星期三 11:06
HTML 实体 插入特殊 HTML 字符实体 点击打开 HTML 实体选择面板，如 &copy;、&reg; 等
分页符 插入分页符 点击生成 [========]，用于打印分页控制

扩展语法

图标 / 名称功能描述用法说明
图表 下拉 插入 Apache ECharts 交互式图表 下拉选择图表类型（柱状图、折线图、饼图、雷达图、漏斗图），自动生成 ```echarts 代码块
标签页 插入 Tabs 标签页组件 选中文字后点击，自动生成 [[tabs]] ... [[/tabs]] 语法
多列排版 插入多栏报纸式排版 选中文字后点击，自动生成 [[columns:N]] ... [[/columns]] 语法
悬浮提示 插入 Tooltip 悬浮提示 支持链接 [文本](tooltip:内容) 和图片 ![alt](url "tooltip:内容")

视图操作

图标 / 名称功能描述用法说明
跳转行 快速跳转到指定行号 点击弹出对话框输入行号，自动定位光标
实时预览 开启或关闭实时预览 点击切换编辑区与预览区的同步显示状态
全窗口预览 进入全窗口预览模式 点击后隐藏编辑区，仅显示预览内容，按 ESC 退出
全屏 编辑器进入全屏模式 点击后编辑器占据整个浏览器窗口，再次点击退出
清空 清空编辑器全部内容 点击后清空编辑区，操作前请确认已保存重要内容
搜索替换 打开搜索替换对话框 支持查找、替换、全部替换，支持正则表达式
帮助 打开 Markdown 语法帮助 点击显示常用 Markdown 语法速查表
关于 显示 Editor.md 关于信息 点击显示版本号、作者和项目链接

下拉菜单说明

Editor.md v1.7.0 将部分功能按钮整合为下拉菜单组，使工具栏更加简洁。下拉菜单默认启用，可通过 toolbarDropdown: false 关闭。

Insert 插入菜单

包含 代码、预格式化文本、代码块、表格、日期时间、HTML 实体、分页符 共 7 个功能。点击 图标展开。

Align 对齐菜单

包含 左对齐、居中对齐、右对齐、两端对齐 共 4 个功能。点击 图标展开。

Chart 图表菜单

包含 柱状图、折线图、饼图、雷达图、漏斗图 共 5 个功能。点击 图标展开。

H1 标题菜单

包含 H1 ~ H6 六级标题。点击 H1 图标展开。
